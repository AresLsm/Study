# ES6에서의 순회와 iterable, iterator protocol

<h2>ES6에서의 List 순회</h2>

- ES6 이전의 리스트 순회는 아래와 같이 했다.

```js
const list = [1, 2, 3];
for (var i = 0; i < list.length; i++) {
  console.log(list[i]);
}

const str = "abc";
for (var i = 0; i < str.length; i++) {
  console.log(str[i]);
}
```

- ES6가 등장하며 아래와 같이 순회할 수 있게 되었다.

```js
const list = [1, 2, 3];
for (const a of list) {
  console.log(a);
}

const str = "abc";
for (const a of str) {
  console.log(a);
}
```

- 코드가 매우 간결해지고, 보다 선언적으로 순회할 수 있게 되었다.
